# Linux Fundamentals Part 1

## Summary
Basics of Linux navigation, file operations, searching, and shell operators.

## Commands
- `echo` - print text  
- `whoami` - show current user  
- `ls` - list files  
- `cd` - change directory  
- `cat` - view file contents  
- `pwd` - show full path  

## Searching
- `find -name <file>` - search for a file  
- `*` - wildcard  
- `grep "<text>" <file>` - search inside files  

## Shell Operators
- `&` - run in background  
- `&&` - chain commands  
- `>` - redirect output  
- `>>` - append output
